

# ESA Pages 个人展示网站

## 项目介绍
本项目是一个**个性化个人展示平台**，围绕“个人技能、经历与技术思考”搭建，核心特点如下：
- **实用性**：作为个人技术名片，可快速向他人展示技能栈、项目经历与学习规划，适配个人求职、技术交流等场景；
- **创意性**：采用模块化布局（个人介绍、技术感悟、未来规划），结合简洁的交互设计，兼顾信息清晰性与视觉体验；
- **技术深度**：基于HTML、CSS、JavaScript前端“三件套”开发，实践了前端页面结构规范、样式布局适配、静态资源管理等基础技术，同时完成了跨平台部署（GitHub + 阿里云ESA Pages）的环境适配。
